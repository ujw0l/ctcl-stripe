Accept Stripe Payment with CT Commerce Lite ecommerce platform.

== Description ==

**CTCL Stripe** is an addon for the CT Commerce Lite platform that allows you to accept secure payments through Stripe. With this integration, you can offer your customers a seamless checkout experience directly on your eCommerce site.

**Key Features:**
- Accept payments via Stripe, one of the most trusted payment gateways.
- Secure transactions with SSL and PCI-DSS compliance.
- Easy setup with your Stripe credentials.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/ctcl-stripe` directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the CT Commerce Lite settings panel, and you will find the Stripe settings under the **Billing Tab**.
4. Enter your Stripe Publishable Key and Secret Key.

== Obtaining Your Stripe Credentials ==

1. Log in to your Stripe account at [dashboard.stripe.com](https://dashboard.stripe.com).
2. Navigate to **Developers > API keys** in the sidebar.
3. Here, you’ll find your Publishable Key and Secret Key under the “Standard keys” section.

== Frequently Asked Questions ==

= How do I add Stripe payment option to my checkout? =

Simply install and activate the CTCL Stripe addon, then enter your Stripe API credentials in the CT Commerce Lite settings under the **Billing Tab**.

= Is Stripe secure? =

Yes, Stripe is one of the most secure payment gateways available, offering SSL and PCI-DSS compliance to ensure your transactions are safe.
